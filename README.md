# Orai (WIP)

Only [SFML](https://github.com/SFML/SFML) dependent Terminal Emulator.

### Run

1. Make `build` dir and in `build` dir run

    ```bash
    cmake .. && cmake --build . && cmake -S ../ -B .
    ```

2. Run from `build` dir

    ```bash
    ./orai
    ```
